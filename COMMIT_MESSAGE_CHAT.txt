feat: add complete real-time chat application with Supabase

This commit adds a production-ready real-time chat application that demonstrates
end-to-end React development with real-time features, authentication, and
modern UI/UX patterns.

Major Features:
- Real-time messaging with Supabase subscriptions
- Multiple chat rooms (general, random, help, tech)
- Online user presence tracking
- Optimistic UI updates for instant feedback
- Message history with auto-scroll
- User profiles with automatic creation
- Responsive design with modern UI

Components Added:
- ChatApp.jsx: Main container with real-time subscriptions and state management
- MessageList.jsx: Displays messages with avatars, timestamps, and grouping
- MessageInput.jsx: Message creation with optimistic updates and keyboard shortcuts
- ChatHeader.jsx: User information and logout functionality
- UserList.jsx: Real-time online users display with presence indicators

Technical Implementation:
- Real-time subscriptions using Supabase channels
- React Query for data fetching and caching
- Optimistic updates for better UX
- Presence system for online/offline tracking
- Row Level Security (RLS) policies
- Auto-scroll to latest message
- Character counter (1000 max)
- Keyboard shortcuts (Enter to send, Shift+Enter for new line)

Database Setup:
- Created SUPABASE_SETUP.md with complete SQL schema
- Profiles table for user information
- Messages table with room support
- Automatic profile creation on signup
- Indexes for performance optimization
- RLS policies for secure data access

Documentation:
- README.md with learning objectives and architecture
- SUPABASE_SETUP.md with step-by-step database setup
- Comprehensive code comments explaining concepts
- Component-level documentation

Integration:
- Added /chat route to App.jsx (protected route)
- Updated DemoNavigation with chat app link
- Installed date-fns for date formatting
- Integrated with existing authentication system

UI/UX Features:
- Modern, clean interface with Tailwind CSS
- Smooth animations and transitions
- Loading states and error handling
- Responsive design for all screen sizes
- Visual indicators for online users
- Message grouping for better readability
- Timestamp formatting (today, yesterday, date)
